# Nx React Workspace

This repository is an Nx monorepo containing a React (Vite) application and shared libraries,
with enforced architecture via Nx module boundaries.

---

## How to Run

From the workspace root:

# Serve the web app
nx serve web

# Build all projects
nx build web

# Lint a project
nx lint web
nx lint hooks

# Run unit tests
nx test web
nx test hooks
